EupneaOS

EupneaOS is a Fedora based operating system, optimized to run on Chromebooks. It comes with native driver support, Android app support and a custom ChromeOS style desktop.

EupneaOS is still in development. Some features may not be available.

Run a custom boot script and choose between a variety of common Linux distributions. Available distros are modified to work on all Chromebooks.

EupneaOS

The Eupnea Project maintains a custom Linux distribution based on Fedora called EupneaOS. It's designed to imitate the look of ChromeOS and is the recommended way of upgrading your Chromebook.

Compressed disk images (or ISO files) are available on the GitHub Releases page and can be flashed directly to a USB stick.

Depthboot

Another way to breathe new life into your Chromebook is through the use of a powerful boot script. Depthboot (formerly known as Breath) allows you to install common Linux distros on your device.

Currently, the following distributions are supported:

  • Ubuntu
  • Debian
  • Arch Linux
  • Fedora
  • Pop!_OS

Follow the setup described in the documentation to build a Depthboot ISO for your Chromebook.
