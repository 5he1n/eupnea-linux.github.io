Eupnea

A new Linux for Chromebooks

Eupnea bootstraps Linux the way Chromebooks like it. It doesn't rely on ChromeOS or custom firmware and supports all drivers on all x86_64(both AMD and Intel) Chromebooks.

Stock Firmware

Depthcharge, the stock firmware of Chromebooks, is capable of booting a 2-partition USB containing the kernel and root filesystem. Depthcharge does not have the ability to use an initramfs, so all the neccessary features are built into the kernel.

Drivers

Eupnea uses the same forked Linux kernel as ChromeOS. Due to this, all devices supported in ChromeOS work in Eupnea. Eupnea supports: HDMI ports, USB-C ports, touchscreens, touchpads and more. Eupnea also has full audio support for most newer Chromebooks.

Customizable and Stable

Eupnea currently supports Ubuntu, Debian, Arch and Fedora. More distros support is currently in development. Eupnea doesn't "monkey-patch" Linux distros to be able to be chrooted, so it is more stable than alternatives like Crouton.

Gaming

Using Intel GVT-G, it is possible to passthrough the integrated graphics into a Windows virtual machine. Amdgpu and i915 are also fully supported in Eupnea.
